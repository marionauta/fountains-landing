Host your own city

Water Finder uses a thin API bewteen the clients and OpenStreetMap. This API processes the data and provides a cache, so the clients don't hit the OpenStreetMap servers too hard.

Right now, each city is hosted in a separate server. If you want to make your city available you can host the API for your city yourself. You can find the code and instructions in the Water Finder backend repository.

You can keep that server for your own personal use, or make it public and share it with the rest. You can send me the address to your server and I will add it to the known servers list. In future versions. the discovery service will be automatic.
